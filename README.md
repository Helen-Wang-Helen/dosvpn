### 极速、简单、开源

dosvpn 是一款方便访问 google、github、Stack Overflow、codepen 等网站的工具，仅用于编程学习与研究。它支持 windows、macOS、linux，未来可能支持 android、ios。灵活的代理模式（全局、智能、自定义），多协议支持（ss、ssr、vmess、http、dns 等）通过对互联网公开节点的筛选和测速，一键连接，方便快捷。
